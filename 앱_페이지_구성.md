# 사주 앱 페이지 구성 및 기능 명세

## 전체 앱 구조

### 앱 흐름도
```
스플래시 화면
    ↓
로그인 여부 확인
    ↓
├─ (로그인 안됨) → 온보딩 → 로그인/회원가입
└─ (로그인 됨) → 메인 화면 (홈)
```

## 페이지별 상세 구성 (총 12개 화면)

---

## 1. 인증 관련 (3개 화면)

### 1.1 스플래시 화면
- **목적**: 앱 로딩 및 초기화
- **구성요소**:
  - 앱 로고
  - 로딩 인디케이터
- **기능**:
  - 로그인 상태 확인
  - 필요 데이터 미리 로드
- **다음 화면**: 온보딩 또는 홈

### 1.2 온보딩 화면 (최초 1회)
- **목적**: 앱 소개 및 주요 기능 안내
- **구성요소**:
  - 3~4개 슬라이드 (스와이프)
  - "시작하기" 버튼
- **내용**:
  1. "재회, 언제 다시 만날 수 있을까요?"
  2. "정확한 만세력 기반 사주 분석"
  3. "매일 업데이트되는 오늘의 운세"
- **다음 화면**: 로그인/회원가입

### 1.3 로그인 & 회원가입 화면
- **구성요소**:
  - 이메일/비밀번호 입력
  - "로그인" 버튼
  - "회원가입" 버튼
  - 소셜 로그인 (선택)
    - 카카오 로그인
    - 구글 로그인
  - "비밀번호 찾기" 링크

#### 회원가입 시 추가 입력
1. **기본 정보**
   - 이메일
   - 비밀번호 (8자 이상)
   - 비밀번호 확인

2. **사주 정보** (필수)
   - 생년월일 (양력/음력 선택)
   - 출생 시간 (선택)
   - 성별
   - 이름 (선택)

---

## 2. 메인 탭 (4개)

### 2.1 홈 탭
- **목적**: 오늘의 운세 및 주요 기능 빠른 접근
- **구성요소**:
  
  **상단**
  - 사용자 이름
  - "안녕하세요, OOO님"
  - 알림 아이콘
  
  **오늘의 운세 카드** (무료)
  - 오늘 날짜
  - 오늘의 총운 (별점 ⭐⭐⭐⭐☆)
  - 오늘의 연애운
  - 한 줄 메시지
  - "자세히 보기" 버튼
  
  **빠른 메뉴 (그리드 2x2)**
  - 🔮 재회 사주 보기 (유료 990원)
  - 💕 만남 시기 보기 (유료 990원)
  - 💑 궁합 보기 (유료 1,900원)
  - 📅 월간 운세 (무료)
  
  **배너**
  - 프로모션 배너 (선택)
  - "첫 구매 50% 할인"

### 2.2 사주풀이 탭
- **목적**: 내 사주 정보 확인 및 심화 분석
- **구성요소**:
  
  **내 사주 카드**
  - 년주: 甲子 (갑자)
  - 월주: 乙丑 (을축)
  - 일주: 丙寅 (병인)
  - 시주: 丁卯 (정묘)
  - 생년월일 (음력)
  - 절기 정보
  
  **관계 및 신살**
  - 반합: 인오반합 ✅
  - 원진살: 자미원진 ⚠️
  - 천을귀인: 술해 ✨
  
  **분석 메뉴**
  - "재회 가능성 보기" (유료)
  - "만남 시기 보기" (유료)
  - "내 사주 자세히 알아보기" (무료)

### 2.3 히스토리 탭
- **목적**: 과거 조회 내역 확인
- **구성요소**:
  - 날짜별 리스트
  - 각 항목:
    - 조회 유형 (재회/만남/궁합)
    - 날짜
    - 결과 요약 1줄
    - "다시 보기" 버튼
  
  **필터**
  - 전체 / 재회 / 만남 / 궁합
  
  **빈 상태**
  - "아직 조회 내역이 없어요"
  - "첫 사주 보러가기" 버튼

### 2.4 마이 페이지
- **목적**: 설정 및 계정 관리
- **구성요소**:
  
  **프로필**
  - 이름
  - 이메일
  - 생년월일
  - "프로필 수정" 버튼
  
  **구매 내역**
  - 구매 일시
  - 항목 (재회 사주)
  - 금액
  
  **설정**
  - 푸시 알림 설정
  - 버전 정보
  - 약관 및 정책
    - 이용약관
    - 개인정보처리방침
  - 고객센터
    - FAQ
    - 1:1 문의
  
  **로그아웃**
  - "로그아웃" 버튼

## 3. 기능별 상세 화면 (5개)

### 3.1 재회 사주 상세 화면 (유료 990원)
- **접근**: 홈 > 재회 사주 보기
- **결제 전 화면**:
  - 서비스 설명
  - "어떤 내용을 확인할 수 있나요?"
    - ✓ 재회 가능성 분석
    - ✓ 최적의 연락 시기
    - ✓ 재회를 위한 조언
  - 가격: 990원
  - "결제하고 보기" 버튼
  - "취소" 버튼

- **결제 후 결과 화면**:
  
  **재회 가능성**
  - 퍼센트 바 (예: 75%)
  - 한 줄 요약
  
  **상세 분석**
  - 현재 상황 분석
  - 상대방의 마음 상태
  - 시기별 재회 운
  
  **최적의 시기**
  - 📅 2025년 3월 중순
  - 구체적인 날짜: 3월 15일 ~ 3월 20일
  
  **조언**
  - 어떻게 접근하면 좋은지
  - 주의할 점
  
  **액션**
  - "캘린더에 저장" 버튼
  - "공유하기" 버튼 (선택)
  - "다운로드" 버튼

### 3.2 만남 시기 화면 (유료 990원)
- **접근**: 홈 > 만남 시기 보기
- **결과 화면**:
  
  **언제 만날까요?**
  - 가장 좋은 시기: 2025년 4월
  - 만남 가능성 타임라인 (월별 그래프)
  
  **어떤 인연을 만날까요?**
  - 나이대: 28~32세
  - 성격: 차분하고 성실한 타입
  - 직업군: 전문직 또는 사무직
  - 만날 장소: 지인 소개, 취미 활동
  
  **월별 만남운**
  - 1월: ⭐⭐☆☆☆
  - 2월: ⭐⭐⭐☆☆
  - 3월: ⭐⭐⭐⭐☆
  - 4월: ⭐⭐⭐⭐⭐ (최고)
  
  **조언**
  - 이 시기에 어떻게 행동하면 좋은지

### 3.3 궁합 보기 화면 (유료 1,900원)
- **접근**: 홈 > 궁합 보기
- **입력 화면**:
  - "상대방 정보를 입력해주세요"
  - 생년월일 (양력/음력)
  - 출생 시간 (선택)
  - 성별
  - "궁합 보기" 버튼

- **결과 화면**:
  
  **궁합 점수**
  - 총점: 85/100 ⭐⭐⭐⭐☆
  - 연애 궁합: 90점
  - 결혼 궁합: 80점
  - 재회 궁합: 75점
  
  **상세 분석**
  - 둘의 천간/지지 관계
  - 반합/충/형/해 관계
  - 장점과 단점
  
  **조언**
  - 갈등이 생길 수 있는 포인트
  - 서로 보완하는 방법
  - 좋은 시기와 조심할 시기

### 3.4 월간 운세 화면 (무료)
- **접근**: 홈 > 월간 운세
- **구성요소**:
  - 이번 달 (12월)
  - 총운: ⭐⭐⭐⭐☆
  
  **주간별 운세**
  - 1주차: 길일 (吉日)
  - 2주차: 평일 (平日)
  - 3주차: 흉일 (凶日) 주의
  - 4주차: 대길 (大吉)
  
  **분야별 운세**
  - 💕 연애운: "새로운 만남 가능성"
  - 💰 재물운: "안정적인 수입"
  - 💼 직업운: "승진 기회"
  - 🏥 건강운: "컨디션 관리 필요"
  
  **이번 달 조언**
  - 좋은 날: 12월 5일, 12일, 18일
  - 조심할 날: 12월 9일, 23일

### 3.5 알림 화면
- **접근**: 홈 > 알림 아이콘
- **구성요소**:
  - 날짜별 알림 리스트
  - 알림 유형:
    - 📅 오늘의 운세 업데이트
    - 💕 재회 좋은 날 알림
    - 🎉 프로모션/할인 소식
    - 💬 공지사항
  
  **빈 상태**
  - "새로운 알림이 없어요"

## 앱 플로우 예시

### 신규 사용자
```
앱 설치
→ 스플래시
→ 온보딩 (3개 슬라이드)
→ 회원가입
   → 이메일/비밀번호 입력
   → 생년월일, 시간, 성별 입력
→ 홈 화면
→ 오늘의 운세 확인 (무료)
→ "재회 사주 보기" 클릭
→ 결제 (990원)
→ 재회 사주 결과 확인
```

### 재방문 사용자
```
앱 실행
→ 스플래시
→ 홈 화면 (자동 로그인)
→ 오늘의 운세 자동 업데이트
→ 히스토리에서 과거 결과 다시 보기
```
